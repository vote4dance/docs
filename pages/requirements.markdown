# Technical requirements

## Devices

Minimum recommended setup:

- Tablets or phones for judges and floor functions on stable Wi-Fi
- One scrutineer computer running Google Chrome for Manager
- Arena display computer(s) for screen output

Optional but recommended:

- Printer for physical reports, diplomas, or start numbers

## Network and internet

Vote4Dance is cloud-based and requires reliable internet during operations.

Checklist:

- Stable internet available for the full event
- Wi-Fi with no forced timeout/logoff behavior
- No captive portal or browser-based login requirement
- Device capacity sized for all event devices
- Backup internet path (secondary router or 4G/5G)
- A designated technical contact on event day

## Browser guidance

For Manager and admin workflows, use current Google Chrome.

## Operational readiness checklist

Before doors open:

1. Test login and permissions for organizer roles
2. Verify judging devices can reach the judging app
3. Verify Speaker and Screens can load event data
4. Run one full test round (ready -> closed -> confirmed -> published)
