# Special formats

These guides cover format-specific setup that extends the standard competition flow.

Use them after you are familiar with the core Manager workflow.

- [Battle bracket](/battle-bracket/) — 1v1 elimination brackets: placing round, seeding, B‑final, and per-size recipes from 2 to 32+ competitors.
- [Jack & Jill](/jack-n-jill/) — random-partner format with callbacks and finals.
- [Point summary](/point-summary/) — combined point totals across multiple rounds or classes.
