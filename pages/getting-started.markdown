# Getting Started

Use this section to get your first event running quickly and safely.

## Recommended path

1. **First time here?** Start with [Understanding Vote4Dance](/understanding-vote4dance/) to learn the system
2. [Quickstart](/quickstart/) — Step-by-step event setup
3. [Technical requirements](/requirements/) — Devices, internet, and browser setup
4. [Vote4Dance building blocks](/building-blocks/) — Overview of all apps

## What you should have after this section

- A created event in Manager
- At least one competition and class
- A usable schedule with rounds
- Judges and teams assigned
- A clear plan for live operations
