# Federation age transfer governance

Age transfer is governed at federation level because it affects eligibility, fairness, and long-term progression.

## What this step is for

This step defines how the federation sets and explains age transitions between seasons and categories.

## How it works

1. Define the age rule reference point.
2. Publish age-group boundaries clearly.
3. Handle edge cases consistently.
4. Support organizers and schools when disputes arise.
5. Keep the federation interpretation stable during the season.

## Common blockers

- Rules are published too vaguely.
- Edge cases are solved ad hoc.
- Different admins communicate different answers.

## What success looks like

Age-group transitions are predictable, fair, and consistently applied.
