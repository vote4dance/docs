# Scrutineer account creation

This is the first step for a scrutineer. It makes sure event access starts from a clean personal account.

## What this step is for

The account identifies the scrutineer for event assignments, permissions, and operational accountability.

## How it works

1. Create a personal Vote4Dance account.
2. Complete your profile details.
3. Confirm which organization or federation context applies.
4. Make sure event assignments use the correct account.

## What success looks like

The scrutineer can be assigned clearly and does not depend on shared credentials.
