# Scrutineer event assignment

Scrutineers are typically assigned rather than self-registering like dancers. This step covers how event work becomes official.

## What this step is for

This step explains how organizers and scrutineers align on event access and operational responsibilities.

## Where to prepare

- [Manager Guide](/manager-guide/)
- [Special formats](/special-formats/)

## How it works

1. Receive the assignment from the organizer.
2. Confirm the event details and operational role.
3. Verify access before event day.
4. Review any special format or ranking-sensitive expectations.

## What success looks like

The scrutineer can enter the event workflow early and is ready before competition day.

## Best pre-event checklist

1. confirm event access works
2. confirm expected formats and panels
3. confirm who resolves issues during the event
4. confirm how result corrections are escalated
