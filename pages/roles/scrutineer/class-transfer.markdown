# Scrutineer class transfer impacts

Class transfer disputes can affect who appears in results and which outcomes are valid.

## What this step is for

This step explains how scrutineers should react when a class-level question appears during event operations.

## How it works

1. Detect suspicious class placement issues.
2. Pause and escalate when needed.
3. Follow the organizer or federation decision.
4. Keep the event record consistent with the approved decision.

## What success looks like

The scrutineer supports a correct outcome without overruling federation policy.
