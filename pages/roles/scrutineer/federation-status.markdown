# Scrutineer federation status through school or club

The school, club, or federation admin usually handles the federation-side joining step. For the scrutineer, this is about confirming the status is in place.

## What this step is for

This step makes sure the scrutineer is recognized correctly inside the federation workflow before event assignment.

## How it works

1. Confirm the right federation applies.
2. Confirm the organization relationship is valid if needed.
3. Confirm the required license is active.
4. Resolve any approval gaps before event work begins.

## What success looks like

The scrutineer can accept assignments without federation status surprises.
