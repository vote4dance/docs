# Scrutineer retiring from active duty

Retirement for a scrutineer means stepping away from assignments while leaving a clean handover.

## What this step is for

This step helps scrutineers end active work without leaving events or organizations dependent on unavailable people.

## How it works

1. Stop accepting new assignments.
2. Resolve any current event commitments.
3. Hand over useful process knowledge where needed.
4. Confirm there is no lingering operational dependency.

## What success looks like

The scrutineer can step away cleanly and the event ecosystem remains stable.
