# Scrutineer

**You are a scrutineer if** you are an official or official's representative responsible for verifying judging accuracy, handling scoring disputes, resolving appeals, or managing ranking quality for a federation or event.

This role hub covers:
- Setting up access and permissions
- Understanding when you're assigned to events or rankings
- Reviewing scores and results for accuracy
- Handling exceptions and appeals
- Managing your role transitions

**Not sure if you're a scrutineer?** Start with [Finding Your Role](/understanding-vote4dance/finding-your-role/).

**Most scrutineer work** happens in the feature sections below after an event is underway.

## How to use this hub

1. Start on a scrutineer step when the issue is about readiness, assignment, or escalation.
2. Switch to [Manager Guide](/manager-guide/) for the operational event workflow.
3. Switch to [Special formats](/special-formats/) when the event format changes the normal flow.
4. Use [Manual Workflows](/federation-rankings/manual-workflows/) only when federation staff have approved an exception that affects the published outcome.
