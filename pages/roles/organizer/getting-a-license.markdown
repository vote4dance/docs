# Organizer getting a license

Some federations require organizers or key functionaries to hold a valid license before they can manage federation-regulated events.

## What this step is for

This step helps you determine whether an organizer license is required and, if yes, get the correct license active before event setup starts.

## Where to go

Use one of these paths depending on federation policy:

- Organizer applies directly: [Dancer self-service](/federation-licenses/dancer-self-service/)
- School/club applies for you: [Organization admin](/federation-licenses/organization-admin/)
- Federation manually issues/approves: [Federation admin](/federation-licenses/federation-admin/)

## Before you start

Confirm these items first:

1. Which federation governs your event
2. Whether organizer licensing is required or exempt
3. Whether your path is self-service, organization-managed, or federation-managed
4. Which season year applies
5. Whether payment is required

## Step-by-step

### Step 1: Confirm if organizer license is required

Ask your federation admin:

- "Do event organizers need a license for this season?"
- "If yes, which license item name should I apply for?"

Possible answers:

- Required: continue to Step 2
- Exempt: save written confirmation and skip to final verification

### Step 2: Choose the correct application path

Use the federation's required model:

| Model | Who starts application | Typical case |
|---|---|---|
| Self-service | Organizer | Independent events |
| Organization-managed | School/club admin | Organizer works under a club/school |
| Federation-managed | Federation admin | Manual vetting workflows |

If you are unsure, do not submit yet. Wrong-path applications are a common blocker.

### Step 3: Submit the application

If self-service:

1. Sign in to Vote4Dance
2. Go to **Account -> Licenses**
3. Click **Apply for License**
4. Select federation and organizer license item
5. Set season year
6. Submit

If organization-managed:

1. Ask organization admin to apply on your behalf
2. Confirm they selected your organizer item and season
3. Confirm your account is linked to that organization

### Step 4: Complete payment (if required)

If **Pay now** appears:

1. Click **Pay now**
2. Complete Stripe checkout
3. Return to Vote4Dance
4. Confirm status changed to paid/pending approval

If no payment appears, continue to approval checks.

### Step 5: Confirm approval status

Check your license row in **Account -> Licenses**:

- Active: done
- Pending approval: wait for federation decision
- Rejected: open reason and correct application

### Step 6: Verify organizer access

Before doing event work:

1. Open Manager
2. Open your event context
3. Confirm you can edit (not read-only)
4. Confirm no license block warning appears

## Common blockers and fixes

### "I paid, but status is still pending"

Cause: manual federation approval required.

Fix:

1. Check expected SLA with federation
2. Send application ID to federation admin
3. Do not create duplicate applications

### "I cannot find organizer license item"

Cause: wrong federation selected or item inactive.

Fix:

1. Re-check federation context
2. Ask federation to confirm exact item name
3. Retry application

### "Organization was supposed to apply, but I applied directly"

Fix:

1. Ask federation whether direct application can be reused
2. If not, cancel direct application
3. Re-apply through organization admin path

## What success looks like

You have either:

- an **Active** organizer license for the correct federation and season, or
- a written federation exemption confirmation

and Manager event editing is unblocked.
